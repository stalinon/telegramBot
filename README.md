# Телеграм-Бот
- Коммуникация с пользователем через интерфейс на WPF (пользователь должен отправить первое сообщение, чтобы программа получила его id);
- При отправке пользователем сообщения с фотографией, изображение обрабатывается (добавляется рандомная подпись) и возвращается пользователю. При отправке команды `/добавить <ваша_подпись>`, введенная подпись будет добавлена в память и следующее обработанное фото будет подписано ей;
- Есть возможность добавлять эти подписи из интерфейса (сериализуются в файл);
- Сохраняет chatID однажды написавших, возможно выбрать никнеймы таковых в ComboBox и отправлять им сообщения из окна.
Обработка изображений проводится с помощью библиотеки ImageProcessor (https://github.com/JimBobSquarePants/ImageProcessor).
